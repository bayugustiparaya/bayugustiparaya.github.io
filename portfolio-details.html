MaMoney (e-Wallet Web-based)

Project information

  • Category: Website. Application Sistem
  • Built: Laravel Framework, Postgre SQL, Firebase storage
  • Project URL:mamoney.herokuapp.com

MaMoney is an e-Wallet application that can be used for various non-cash and non-card transactions. This application is web-based. The feature is user can set spending limits. It has the choice of saving or not before transaction, this aims to increase the desire to save.

MaMoney was deploy on heroku. It's built with Laravel Framework, Postgre SQL and image is stored on Firebase Storage. Have a friendly UI and it use barcode generate automatic.
